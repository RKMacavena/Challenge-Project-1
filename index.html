Using CSS selectors and Specificity

What are CSS selectors?

CSS selectors are used to target elements in HTML to apply styles to them. They make use of tags and attributes in HTML. Below are the selectors, names and descriptions.

SELECTOR NAME DESCRIPTION
p tag Targets elements by using the tags of elements
* universal Targets all HTML elements
.class class Targets elements by using the value of the class attribute
#id id Targets elements by using unique values of id attribute
[img]

img[src*='value']		 attributes Targets elements that contain attributes in the opening tag.
It can also target specific elements using a unique value from the url from the attribute.
p:hover pseudo Targets elements using the tags of elements with styles that are only visible by user interaction
p.class chain Targets elements by combining different selectors
.class li
or
#id li		 descendent Targets elements nested in other elements that have class or id attributes
p, body combination Targets elements that are unrelated using their tags

What is specificity?

Elements can be styled using different selectors. Specificity determines which style is displayed by the browsers by the selector used. id selectors are more specific than class selectors which in turn are more specific than tag selectors. "!important" can be used at the end of a declaration that overrides any specificity.
